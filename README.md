## Block Web

Clone the project

```bash
  git clone https://github.com/salarrbl/fun
```

Go to the project directory

```bash
  cd fun/Web-Blocker
```

block web-site

```bash
  sudo ./blocker.sh block www.xnxx.com
```

unblock

```bash
  sudo ./blocker.sh unblock www.xnxx.com
```
